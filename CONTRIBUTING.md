# Contributing to GTB DLP Policies & Automation

Thanks for your interest!

## How to Contribute
1. Fork the repo
2. Create a branch (`git checkout -b feat/my-policy`)
3. Add your policy/script
4. Commit with clear messages
5. Open a Pull Request

## Code Style
- YAML: 2-space indent
- Python: PEP 8
- All policies must be demo-safe

Questions? Open an issue.
